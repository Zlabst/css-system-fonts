System Fonts for iOS

This demo is for iOS 9+

"iOS has dynamic type behavior, which can provide an additional level of fit and finish to your content. These text styles identify more than simply a particular font family; instead, they represent an entire style, including size and weight. These styles are therefore characterized by values given to the more-general font CSS property."

- Using the System Font in Web Content
